# Hospitality & Event Booking SaaS Starter

## ใช้ 4 อย่าง
1. GitHub Pages
2. Google Sheets
3. Google Apps Script
4. Cloudinary

## วิธีเริ่ม
1. เปิดไฟล์ `Table_Event_Booking_Google_Sheets_Template.xlsx` ด้วย Google Sheets
2. Copy Spreadsheet ID
3. เปิด Apps Script แล้ววาง `apps-script/Code.gs`
4. ใส่ Spreadsheet ID ใน `SPREADSHEET_ID`
5. Deploy เป็น Web App
6. Copy Web App URL ไปใส่ `assets/js/config.js`
7. อัปทั้งโฟลเดอร์นี้ขึ้น GitHub Pages

## URL
- ลูกค้า: `/index.html`
- แอดมิน: `/admin.html`
